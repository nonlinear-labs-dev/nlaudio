Detailed Description

Pure virtual base class for all audio classes.

This defines the interface for all audio classes, such as:

  • Nl::AudioAlsaInput
  • Nl::AudioAlsaOutput
  • Nl::AudioJackInput
  • Nl::AudioJackOutput
